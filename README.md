# PostgreSQL to Azure Migration Proposal Generator

A professional web application that generates comprehensive migration proposals for moving PostgreSQL databases to Azure Database for PostgreSQL.

## Features

- **Interactive Assessment Form**: Collect detailed information about current PostgreSQL infrastructure
- **Automated Recommendations**: Get Azure Database tier recommendations based on your workload
- **Cost Analysis**: Compare current costs with projected Azure costs
- **Migration Roadmap**: Detailed timeline with phases and milestones
- **Professional Output**: Generate polished proposals ready for stakeholder review
- **Export Options**: Download as PDF or print directly

## Usage

Simply open the application in your browser and:

1. Fill out the **Database Assessment** form with your current infrastructure details
2. Provide **Business Information** including company name and preferred Azure region
3. Click **Generate Proposal** to create a comprehensive migration proposal
4. Review the generated proposal with executive summary, cost analysis, and migration roadmap
5. Download as PDF or print for distribution

## Technology Stack

- **HTML5** - Semantic structure
- **CSS3** - Modern styling with custom properties and gradients
- **Vanilla JavaScript** - Interactive form handling and proposal generation
- **html2canvas & jsPDF** - PDF export functionality
